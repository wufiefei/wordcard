# 🎨 宝宝单词闪卡生成器

一个美观、易用的网页应用，可以用宝宝的照片生成个性化的单词学习闪卡，支持打印和下载。

## ✨ 功能特点

### 已实现功能

1. **照片上传** 📸
   - 支持拖拽上传和点击上传
   - 支持 JPG、PNG 等常见图片格式
   - 实时预览上传的照片
   - 友好的上传交互体验

2. **单词库管理** 📚
   - 多个预置单词库（动物、水果、颜色等）
   - 每个单词库包含 10-20 个常用单词
   - 每个单词包含：
     - 英文单词
     - 中文翻译
     - 自然拼读标注（不同颜色）
     - 卡片背景图片位置
     - 人脸定位信息

3. **自然拼读颜色系统** 🎨
   - 短元音（Short Vowels）：红色
   - 长元音（Long Vowels）：蓝色
   - 辅音（Consonants）：黑色
   - 双辅音/三辅音（Digraphs & Trigraphs）：橙色
   - 不发音字母（Silent）：灰色
   - R控制元音（R-controlled vowels）：紫色
   - 双元音（Diphthongs）：粉色

4. **卡片尺寸选择** 📐
   - 超大卡：1张/页（150×210mm）- 幼儿启蒙
   - 大卡：2张/页（110×165mm）- 图片为主
   - 中卡：6张/页（80×100mm）- 默认推荐 ⭐
   - 标准卡：8张/页（63×88mm）- 经济实用
   - 方形卡：9张/页（100×100mm）- ins风

5. **单词卡片选择** ✅
   - 缩略图方式展示所有单词
   - 支持单个选择和全选
   - 显示已选择数量
   - 点击放大预览单词卡效果

6. **卡片预览** 👀
   - 弹窗查看单词卡详细效果
   - 显示自然拼读颜色
   - 预览宝宝头像位置（如已上传照片）
   - 实时查看卡片布局

7. **导出功能（占位）** 🎁
   - PDF导出按钮（功能待开发）
   - 单张图片导出按钮（功能待开发）
   - 排版预览按钮（功能待开发）

8. **美观的UI设计** 🌈
   - 可爱、小清新的配色方案
   - 渐变色和圆角设计
   - 响应式布局，适配移动端
   - 流畅的动画过渡效果
   - 自定义滚动条样式

## 🎯 使用方法

1. **上传宝宝照片**
   - 点击或拖拽照片到上传区域
   - 支持 JPG、PNG 格式
   - 上传后可预览照片

2. **选择单词库**
   - 点击想要学习的单词库（动物、水果、颜色等）
   - 单词库卡片会显示单词数量

3. **选择卡片尺寸**
   - 根据使用场景选择合适的卡片尺寸
   - 默认推荐使用"标准卡"（6张/页）

4. **选择要生成的单词**
   - 单击单词卡进行选择
   - 点击"全选"按钮选择所有单词
   - 点击单词卡可放大预览效果

5. **导出卡片**
   - 查看已选择的卡片数量
   - 点击"导出为PDF"用于打印
   - 或点击"导出单张图片"用于电子展示
