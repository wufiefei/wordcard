# 📖 使用指南

## 🎯 产品介绍

宝宝单词闪卡生成器是一个帮助家长为宝宝制作个性化英语学习闪卡的网页工具。通过上传宝宝的照片，选择合适的单词库，就能生成带有宝宝头像的可爱单词卡片，让英语学习更有趣、更亲切！

## 🚀 快速上手（5步搞定）

### 第1步：上传宝宝照片 📸

1. 在左侧找到"上传宝宝照片"区域
2. 可以选择：
   - **点击上传**：点击"选择照片"按钮，从电脑中选择照片
   - **拖拽上传**：直接将照片拖到上传区域
3. 支持的格式：JPG、PNG等常见图片格式
4. 上传成功后会显示照片预览

**💡 小提示**：
- 建议使用正面清晰的宝宝照片
- 人脸占比适中效果最佳
- 如需更换照片，点击"重新选择照片"按钮

### 第2步：选择单词库 📚

在"选择单词库"区域，点击想要学习的主题：

- **🐾 动物** - 20个常见动物单词（cat、dog、bird等）
- **🍎 水果** - 20个常见水果单词（apple、banana、orange等）
- **🎨 颜色** - 11个基础颜色单词（red、blue、yellow等）

选中的单词库会高亮显示，右侧会显示该库的所有单词。

### 第3步：选择卡片尺寸 📐

根据使用需求选择合适的卡片尺寸：

| 类型 | 推荐场景 | 特点 |
|------|---------|------|
| 超大卡 | 幼儿启蒙 | 2张/页，最大最清晰 |
| 大卡 | 图片为主学习 | 4张/页，视觉冲击强 |
| **标准卡** ⭐ | 日常使用 | 6张/页，平衡性最好（推荐） |
| 小卡 | 经济实用 | 8张/页，省纸环保 |
| 方形卡 | ins风格 | 9张/页，正方形可爱 |
| 迷你卡 | 超便携 | 10张/页，随身携带 |

**💡 小提示**：
- 首次使用建议选择"标准卡"
- 年龄小的宝宝建议用"超大卡"或"大卡"
- 想省纸可以选择"小卡"

### 第4步：选择要生成的单词 ✅

在右侧单词卡片区域：

1. **查看单词**：
   - 每个卡片显示单词的自然拼读（彩色字母）
   - 下方显示中文翻译

2. **选择单词**：
   - 点击卡片左上角的圆圈，选中/取消该单词
   - 选中的卡片会有绿色边框高亮
   - 右上角显示"已选X"

3. **批量操作**：
   - 点击"全选"按钮可选择所有单词
   - 再次点击可取消全选

4. **预览单词卡**：
   - 点击卡片中间区域可放大预览
   - 查看单词卡的完整效果
   - 如已上传照片，可看到宝宝头像位置

### 第5步：生成和导出 🎁

在左侧底部的"生成和导出"面板：

1. **查看统计**：
   - 显示已选择的卡片数量
   - 预估需要的A4纸张数

2. **预览排版**：
   - 点击"👀 预览排版效果"（功能开发中）
   - 查看打印时的整体布局

3. **导出选项**：
   - **📄 导出为PDF**（功能开发中）
     - 适合打印实体卡片
     - 可直接发送到打印机
   
   - **🖼️ 导出单张图片**（功能开发中）
     - 适合在平板、手机上使用
     - 可保存为图片集

## 🎨 自然拼读颜色说明

应用使用不同颜色标注单词的发音规则，帮助宝宝学习：

- 🔴 **红色** - 短元音 (a, e, i, o, u的短音)
- 🔵 **蓝色** - 长元音 (a, e, i, o, u的长音)
- ⚫ **黑色** - 辅音 (b, c, d, f等)
- 🟠 **橙色** - 双辅音/三辅音 (sh, ch, th等)
- ⚪ **灰色** - 不发音字母 (silent e等)
- 🟣 **紫色** - R控制元音 (ar, or, er等)
- 🩷 **粉色** - 双元音 (oi, ou, ow等)

例如：**c**🟢**a**🔴**t**🟢 = cat（猫）

## 💡 使用技巧

### 打印建议
1. 使用300DPI设置打印获得最佳效果
2. 建议使用较厚的纸张（200g以上）
3. 可以过塑保护，延长使用寿命
4. 打印后沿虚线裁切

### 学习建议
1. **循序渐进**：先从小词库开始（如颜色）
2. **每日复习**：每天挑选5-10张卡片复习
3. **游戏化学习**：
   - 找卡片游戏：说单词让宝宝找对应卡片
   - 配对游戏：将单词卡和实物配对
   - 记忆游戏：翻卡片记忆
4. **亲子互动**：一起读单词，加深印象

### 照片选择建议
1. **表情**：选择宝宝笑容灿烂的照片
2. **背景**：纯色背景效果最佳
3. **清晰度**：确保照片清晰，不模糊
4. **光线**：光线充足的照片最佳

## ❓ 常见问题

### Q1: 照片上传后看不到预览？
A: 请检查：
- 图片格式是否为JPG或PNG
- 图片大小是否过大（建议<10MB）
- 浏览器是否允许访问文件

### Q2: 如何更换照片？
A: 在照片预览下方点击"重新选择照片"按钮

### Q3: 选择单词后如何取消？
A: 再次点击该单词卡左上角的选择圈即可取消

### Q4: 可以同时选择多个单词库吗？
A: 目前只能选择一个单词库，未来会支持多库选择

### Q5: 导出的PDF可以编辑吗？
A: 导出的PDF是最终成品，不可编辑

### Q6: 为什么有些功能是灰色的？
A: 这些是高级功能，正在开发中：
- 智能抠图
- PDF导出
- 图片批量下载
- 排版预览

## 🎯 最佳实践

3. 使用"小卡"或"迷你卡"节约纸张
4. 建立学习计划，定期更新卡片
