# 🚀 快速启动指南

## 5分钟开始使用

### 第一步：启动项目 (1分钟)

```bash
# 确保在项目根目录
cd wordcard

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

看到以下输出表示成功：
```
▲ Next.js 15.5.5
- Local:        http://localhost:3000
✓ Ready in 2.5s
```

### 第二步：打开浏览器 (10秒)

在浏览器中访问：**http://localhost:3000**

你会看到一个漂亮的粉紫蓝渐变页面，标题是"✨ 宝宝单词闪卡生成器 ✨"

### 第三步：上传照片 (30秒)

1. 找到左侧第一个卡片"📸 上传宝宝照片"
2. 点击"选择照片"按钮
3. 从电脑中选择一张宝宝照片
4. 上传成功后会显示照片预览

💡 **提示**：也可以直接拖拽照片到上传区域！

### 第四步：选择单词库 (10秒)

1. 在"📚 选择单词库"区域
2. 点击你想要的主题：
   - 🐾 动物（20个单词）
   - 🍎 水果（20个单词）
   - 🎨 颜色（11个单词）

选中的词库会高亮显示。

### 第五步：选择卡片尺寸 (10秒)

1. 在"📐 选择卡片尺寸"区域
2. 推荐选择"标准卡"（默认已选中）
3. 也可以根据需要选择其他尺寸

### 第六步：选择单词 (1分钟)

1. 右侧会显示该词库的所有单词
2. 点击单词卡左上角的圆圈进行选择
3. 或点击右上角"全选"按钮
4. 点击单词卡中间可以放大预览

### 第七步：查看效果 (1分钟)

1. 点击任意单词卡片中间区域
2. 在弹出的预览窗口中查看效果
3. 可以看到宝宝头像的位置和单词的颜色标注

### 第八步：导出（功能开发中）

1. 左侧底部"🎁 生成和导出"面板
2. 显示已选择的卡片数量
3. 点击导出按钮（目前是占位功能）

## 🎯 核心操作一览

| 操作 | 位置 | 快捷方式 |
|-----|------|---------|
| 上传照片 | 左侧第1个卡片 | 拖拽上传 |
| 选择词库 | 左侧第2个卡片 | 单击选择 |

| 预览单词 | 单词卡中间 | 点击放大 |
| 导出卡片 | 左侧第4个卡片 | 点击按钮 |

## 💡 使用技巧

### 技巧1：快速选择
- 先点击"全选"，再取消不需要的单词
- 比一个个选择更快

### 技巧2：多次尝试
- 可以选择不同词库对比效果
- 可以更换不同照片看看效果
- 可以尝试不同卡片尺寸

### 技巧3：预览确认
- 选择完单词后，逐个点击预览
- 确保每张卡片效果满意
- 特别注意宝宝头像位置

### 技巧4：分批生成
- 不用一次选择所有单词
- 可以按主题分批生成
- 每批5-10个单词最合适

## ❓ 常见问题快速解答

### Q: 照片上传后看不到？
A: 刷新页面重试，或检查图片格式（支持JPG/PNG）

### Q: 点击词库没反应？
A: 检查是否已经选中（有紫粉色高亮）

### Q: 如何取消选择的单词？
A: 再次点击该单词卡左上角的圆圈

### Q: 能同时选择多个词库吗？
A: 目前只能选择一个，未来版本会支持

### Q: 导出按钮点击后没反应？
A: 导出功能还在开发中，目前是占位按钮

### Q: 手机上能用吗？
A: 可以！界面已适配移动端

## 📱 移动端使用

### 手机访问方式

1. **同一WiFi下**：
   ```
   手机浏览器访问：http://[你的电脑IP]:3000
   例如：http://192.168.1.100:3000
   ```

2. **查看电脑IP**：
   ```bash
   # Windows
   ipconfig
   
   # Mac/Linux
   ifconfig
   ```

### 移动端操作提示
- ✅ 支持触摸操作
- ✅ 支持拖拽上传
- ✅ 可以直接拍照上传（选择照片时选择拍照）
- ✅ 自动适配小屏幕

## 🚀 性能优化建议

### 浏览器选择
- ✅ 推荐：Chrome/Edge（最佳性能）
- ✅ 支持：Firefox、Safari
- ⚠️ 避免：IE浏览器

### 照片建议
- 📏 尺寸：800×800 ~ 2000×2000
- 💾 大小：< 5MB
- 🎨 格式：JPG（压缩后）或PNG

### 网络建议
- 首次访问需要加载资源
- 后续使用会自动缓存
- 建议在稳定WiFi下使用
